Privacy Policy

What information do we collect?

We collect the SOQL queries executed by the user on https://workbench.developerforce.com/. No other personal data is collected. The features on the login page of the above url just helps to provide a one-click option to login.

How do we use the information

All the collected data is stored within the local storage of the client browser. The extension displays the queries back to the end user in a formatted manner.

What information do we share?

Google Analytics is used to analyze extension usage. Other than that, no data is shared or collected by us. All code is executed on the client browser and does not communicate with any server.
